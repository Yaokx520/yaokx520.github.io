### 📚 [文学气质小测](https://yaokx520.github.io/Literary-personality-test/) · [仓库](https://github.com/Yaokx520/Literary-personality-test)
**测测你更像哪个文学家** — 不是考文学常识，而是看气质、叙事方式与精神底色。
- **66 位**中外文学家 · **36 道**情境题中随机 **10–16 题**
- 六维气质：现实批判 · 抒情浪漫 · 哲思抽象 · 都市孤独 · 人文悲悯 · 先锋实验
- 结果页含 Top 5、雷达图、名言摘记与分享文案
- 提供 **[国际多语言版](https://yaokx520.github.io/Literary-personality-test/international/)**（英 / 日 / 法 / 西 / 葡 / 俄）

### 🌸 [诗心小测](https://yaokx520.github.io/Literary-personality-test/chinese_poet_personality_test.html) · [仓库](https://github.com/Yaokx520/Literary-personality-test)
**测测你更像哪位中国诗人** — 从先秦至清末民初 **66 位**诗人中，寻一位与你诗心相映的古人。
- **36 题**题池，每次随机 **10–16 题**；落红、夜读、江舟、雪夜等诗境情境

### ⚔️ [新三国人格测试](https://yaokx520.github.io/HOMM3/sanguo/) · [仓库](https://github.com/Yaokx520/HOMM3)
**你是《新三国》里的哪位人物** — 这不是历史考试，这是**权谋气味检测器**。
- **16 位**结果人物 · 大题库中随机 **10–18 题** · **20 条**结语判词
- 测权谋、上头、体面、嘴硬、仁义与抽象气质
- 《新三国》戏味长句 + 适合截图发群的狠话短句
- 结果页按人物气质切换氛围 BGM（《关羽之歌》《小乔之歌》《刘备之死》等）
- 支持**直达链接分享**，他人点开即见同一人物结果
